# 💱 Real-Time Currency Converter

## 📌 Description
A Python CLI application that converts currencies in real-time using ExchangeRate API.

## 🚀 Features
- Real-time currency conversion
- Multiple currency support
- Conversion history (JSON)
- API error handling
- Clean CLI interface

## 🔧 Setup

1. Install dependencies:

pip install requests


2. Add API key in config.py

3. Run:

python main.py


## 📊 Menu Options
- Convert Currency
- View Exchange Rates
- View History
- Exit

## 📁 Files
- main.py → Entry point
- api.py → API calls
- converter.py → Conversion logic
- history.py → Save/load history
